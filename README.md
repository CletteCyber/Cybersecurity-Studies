# Cybersecurity Studies - Week 1 
## the CIA Triad
Today I am learning the foundation of Information Security: The CIA Triad.
1.**Confidentiality**: Ensuring that data is accessible only to those authoried to have access. ( Example: Using a password or encryption.)
2.**Integrity**: Ensuring that data is accurate and has not been tampered with or modified by unauthorized users.( Example: Digital signatures.)
3.**Availability**: Ensuring that system and data are available when needed by authorized users. (Example: Protection against power outages or cyber attacks like DDoS).
